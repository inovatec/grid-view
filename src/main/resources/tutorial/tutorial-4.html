Menu Gerenciar > Professores

Nesse menu é possível gerenciar os professores da escola.

Todas os professores cadastrados são mostrados em uma tabela, classificados pelo nome, e-mail e telefone para contato.

É possível cadastrar novos professores por meio do botão Adicionar e verificar os dados de uma determinado professor através do botão Visualizar, desde que um professor seja selecionado na tabela.

Para cada novo professor, deve-se informar os dados pessoais, como nome, rg, cpf, e-mail, telefone, celular e endereço.

Além disso, devem ser informadas as competências do professor por meio do botão Competências presente na tela dos dados do professor. As competências estão relacionadas as disciplinas que o professor está apto a lecionar.

Atalho: [F5]
