Menu Gerenciar > Disciplinas

Nesse menu é possível gerenciar as disciplinas oferecidas pela escola.

Todas as disciplinas cadastradas são mostradas em uma tabela, classificadas pelo nome.

É possível cadastrar novas disciplinas por meio do botão Adicionar e verificar os dados de uma determinada disciplina através do botão Visualizar, desde que uma disciplina seja selecionada na tabela.

Atalho: [F3]
