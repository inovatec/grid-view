Menu Gerenciar > Salas de Aula

Nesse menu é possível gerenciar as salas de aula presentes na escola.

Todas as salas de aula cadastradas são mostradas em uma tabela, classificadas pelo nome, tipo e situação.

É possível cadastrar novas salas de aula por meio do botão Adicionar e verificar os dados de uma determinada sala de aula através do botão Visualizar, desde que uma sala de aula seja selecionada na tabela.

Para cada nova sala de aula, deve-se informar:

  • Nome: Nome da sala de aula.
  • Tipo: Tipo da sala de aula, que pode ser Sala ou Laboratório.
  • Situação: Situação da sala de aula, que indica se a mesma está disponível ou indisponível.
  • Descrição: Uma descrição geral sobre a sala, adicionando alguma outra informação importante.

Atalho: [F4]
